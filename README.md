# ErgoTimer
Web app to auxiliary your productivity time and help your health with pause time to ergonomics exercises.

## How to run

```jsx
git clone https://github.com/gabriellennon/ergotimer.git
```

```jsx
#access folder
cd ergotimer

## Run
yarn or npm install or pnpm install 

## start project
yarn dev
npm run dev
pnpm dev
```
